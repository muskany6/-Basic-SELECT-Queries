Task 3: Writing Basic SELECT Queries

 Objective
To retrieve and filter data from the Student table using SQL SELECT queries.

Tools Used
- Oracle FreeSQL

 Table Used
Student

 SQL Operations Performed

- Displayed all records using SELECT *.
- Retrieved specific columns.
- Filtered records using the WHERE clause.
- Used AND and OR operators to combine conditions.
- Retrieved records using the LIKE operator for pattern matching.
- Selected records within a range using BETWEEN.
- Sorted data using ORDER BY in ascending and descending order.
- Displayed only the first few records using FETCH FIRST (Oracle equivalent of LIMIT).

SQL Concepts Covered
- SELECT
- WHERE
- AND
- OR
- LIKE
- BETWEEN
- ORDER BY
- FETCH FIRST n ROWS ONLY

Outcome
Successfully retrieved, filtered, sorted, and limited data from the Student table using various SQL SELECT queries.
